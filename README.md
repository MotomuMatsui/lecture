# バイオインフォマティクス・機械学習 講義資料

大学初年次の一般教養から学部専門の入口までを想定した、**バイオインフォマティクスと機械学習**の自習用Jupyterノートブック集です。すべての解説文は日本語、コードとデータの識別子は英語で統一されています。

ノートブックは大きく **`ml/`(機械学習編)** と **`phylogenetics/`(系統解析編)** の2系統に分かれており、それぞれ独立に動作します。生物学を出発点として、回帰・分類・クラスタリング・次元削減・タンパク質言語モデル・系統樹推定までを、生き物の具体例を通じて体験できます。

## ディレクトリ構成

```
lecture/
├── ml/                          # 機械学習編(4ノートブック)
│   ├── 1_penguins.ipynb
│   ├── 2_wine.ipynb
│   ├── 3_pLM.ipynb
│   └── 4_gpcr.ipynb
├── phylogenetics/               # 系統解析編(2ノートブック + データ)
│   ├── 1_birds.ipynb
│   ├── 2_mammals.ipynb
│   ├── birds/                   # 31種の鳥類 cytb 配列+メタデータ+PhyloPic
│   ├── mammals/                 # 37種の哺乳類 cytb 配列+メタデータ+PhyloPic
│   └── gpcr/                    # 25種のヒトGPCR 配列+UniProt索引+AlphaFold PDB
└── images/                      # ノートブックから参照される画像素材
```

## 動かし方

各ノートブックを Jupyter または VS Code で開き、上のセルから順に実行してください。各ノートブックの先頭セルに `%pip install …` が書かれており、それがそのノートブック専用の依存リストとなります。リポジトリ全体の `requirements.txt` はありません。

- **`ml/3_pLM.ipynb` と `ml/4_gpcr.ipynb`** は初回実行時に Hugging Face から `Rostlab/prot_bert`(約1.6GB)をダウンロードします。CUDA が使える環境では自動的に GPU を利用します。
- **`ml/2_wine.ipynb`** は `archive.ics.uci.edu` から CSV を直接取得します。
- **`ml/1_penguins.ipynb`** は `palmerpenguins` パッケージ経由でデータを取得します。
- **系統解析編** はリポジトリ内の FASTA ファイル(`phylogenetics/birds/`, `phylogenetics/mammals/`)を直接読み込みます。「本格編」セクションでは `MAFFT` と `IQ-TREE` の外部ツールを別途インストールします(ノートブック内に手順あり)。

> プロットの日本語表示には `matplotlib-fontja` を使用しています。`seaborn` の `sns.set_*` 系を呼ぶと `rcParams` がリセットされるため、必ず **`sns.set_style()` → `import matplotlib_fontja` → `rcParams` 上書き** の順で設定してください。順序を入れ替えると豆腐(□□□)になります。

## ノートブック詳細

### 機械学習編 (`ml/`)

#### 1. `1_penguins.ipynb` — Palmer Penguins で機械学習一巡り

**南極パルマー諸島の3種のペンギン(Adelie / Chinstrap / Gentoo、約340個体)** の形態学的データをもちいて、機械学習の主要トピックを一気に体験する入門ノートブックです。「くちばしの長さ・深さ」「ヒレの長さ」「体重」「島」「性別」などのシンプルな数値・カテゴリ変数だけで、以下を一通り実行します。

- **教師あり学習**
  - 一般化線形モデル(GLM)— ロジスティック回帰で**性別**を予測。`statsmodels` の summary を読み、係数・標準誤差・p値の意味を学ぶ
  - ランダムフォレスト — **種**の3クラス分類と、特徴量重要度の可視化
- **教師なし学習(クラスタリング)**
  - 階層的クラスタリング — デンドログラムを描き、群構造を目で確認する
  - k-means クラスタリング — もう一つの代表的手法
  - グラフクラスタリング — k-NN グラフ上で Louvain 法を走らせるコミュニティ検出
- **次元圧縮による可視化**
  - PCA / MDS(PCoA)/ t-SNE / UMAP の4手法を並べて、線形手法と非線形手法の見え方の違いを比較

**学習のねらい**: 「**解釈可能なモデル(GLM)**」と「**ブラックボックスモデル(Random Forest)**」を同じデータに当てて、係数 vs. 特徴量重要度という両者の説明方法の違いを体感することです。以降の3つのノートブックも、この対比を意識した構成になっています。

#### 2. `2_wine.ipynb` — ワインの化学組成から品質を予測する

**ポルトガル「Vinho Verde」ワインの赤約1,600本・白約4,900本**(UCI Machine Learning Repository)を題材に、ペンギンよりも踏み込んだ手法を実装します。説明変数はpH、揮発性酸度、残糖、アルコール度数など11個の連続値の化学計測値、目的変数は官能評価による品質スコア(0〜10の整数)です。

- **ポアソン回帰(GLM)** — 整数のスコアを直接モデル化する場合の自然な選択肢としてポアソン分布を導入
- **正規化線型回帰(Ridge / Lasso)** — ワインの化学成分間に強い相関(多重共線性)があることを確認し、正則化でそれに対処する
- **XGBoost(勾配ブースティング)** — 決定木のアンサンブルの最終形態。品質予測の精度向上と特徴量重要度の比較
- **ニューラルネットワーク** — 全結合NNを最小構成で組み、損失曲線を眺める。生成AIへの第一歩を踏み出す導入
- **階層的クラスタリング** — ラベル(赤/白)を与えずに、化学組成だけから赤ワインと白ワインが分離できるかを確認
- **主成分分析(PCA)** — 因子負荷量を見て「どの化学変量が主成分に効いているか」を読み解く

**学習のねらい**: 線形 → 正則化 → ツリーアンサンブル → ニューラルネットという**モデル複雑度の階段**を実データの上で踏み、それぞれが「何を解決するために生まれた手法か」を理解することです。

#### 3. `3_pLM.ipynb` — タンパク質言語モデル(ProtBERT)入門

**タンパク質を「生命の言語」として扱う**大規模言語モデル ProtBERT(Rostlab/prot_bert)を使い、アミノ酸配列だけから**膜貫通型タンパク質か可溶性タンパク質か**を識別する二値分類に挑戦します。配列を1024次元の埋め込みベクトルに変換し、そのベクトル空間上で機械学習を行うという、現代のタンパク質情報学の標準ワークフローを体験します。

- **ProtBERT による埋め込み** — 任意長のアミノ酸配列を、固定長(1024次元)の数値ベクトルに変換
- **埋め込み空間の可視化** — PCA / t-SNE / UMAP で2次元に落とし、「類似タンパク質が近くに配置されるか」を確認。膜貫通型(GPCR, チャネル, トランスポーター)と可溶性(酵素, 抗体, ホルモン)が空間内で分かれているかを観察
- **機械学習による分類** — ロジスティック回帰とランダムフォレストの両方で分類モデルを学習。両者の精度を比較
- **新規タンパク質への適用** — 学習に使っていない配列を渡して予測を確認

**学習のねらい**: ProtBERT は「事前学習済み」モデルであり、その埋め込みベクトルさえ手に入れば**シンプルな線形分類器でも高精度に分類できる**という事実を体感することです。原著論文(Elnaggar et al., *IEEE TPAMI* 2022)の核心に触れます。**初回実行時にモデルダウンロード(約1.6GB)** が発生する点に注意。

#### 4. `4_gpcr.ipynb` — GPCR-リガンド親和性予測(創薬への入口)

**G蛋白質連関受容体(GPCR)とペプチドリガンドの相互作用予測**にProtBERT埋め込みを応用するノートブックです。GPCRは全医薬品の約35%が標的とする最重要受容体ファミリーで、本ノートブックは**ペアの配列から結合/非結合を予測する**という、創薬の中核タスクの簡易版を体験します。

- **二本鎖の埋め込み** — GPCR配列とペプチド配列をそれぞれ ProtBERT で埋め込み、両者を結合して相互作用特徴ベクトルを構成
- **二値分類器の学習** — ランダムフォレストと XGBoost で「結合 vs 非結合」を予測
- **モデル比較** — 両者の精度・特徴量重要度・予測の安定性を比較し、創薬スクリーニングへの応用イメージを掴む

**学習のねらい**: `3_pLM.ipynb` で学んだ「単一タンパク質の埋め込み」を「**タンパク質ペアの埋め込み**」へと拡張し、相互作用予測(PPI予測)の枠組みを理解することです。本リポジトリの `phylogenetics/gpcr/` には25種のヒトGPCRの配列とAlphaFold予測構造が同梱されており、結果の生物学的解釈を補助します。

### 系統解析編 (`phylogenetics/`)

#### 5. `1_birds.ipynb` — 鳥類の系統樹を作ろう

**30種の鳥類 + 外群としてのミシシッピワニ、計31種**のミトコンドリア *cytb* 遺伝子配列から、進化的近縁関係を表す**系統樹(phylogenetic tree)** を構築します。同じデータを2通りのアプローチで解析し、入門〜実務までを橋渡しします。

- **入門編(Python のみで完結)**
  - FASTA形式の配列を読み込み、**切り詰めアラインメント**で簡易にアラインを揃える
  - **遺伝的距離**(p-distance、Jukes-Cantor補正など)を計算
  - **近隣結合法(Neighbor Joining; NJ法)** で系統樹を構築
  - PhyloPicの種シルエット画像を樹に重ねて可視化
- **本格編(研究現場の標準ツール)**
  - **MAFFT** で多重配列アラインメント
  - **IQ-TREE** で **最尤法(Maximum Likelihood)** による系統樹推定
  - **ブートストラップ** で各枝の信頼度を評価

**学習のねらい**: 「遺伝的距離」「近隣結合法 vs 最尤法」「ブートストラップによる信頼度評価」という系統解析の三本柱を、鳥類進化の文脈で身につけることです。なぜミトコンドリアの *cytb* 遺伝子が系統解析の定番なのか(保存性・母系遺伝・約1140塩基という扱いやすい長さ)も学びます。

#### 6. `2_mammals.ipynb` — 海に戻った哺乳類の系統樹

**37種の哺乳類**(ヒゲクジラ亜目7種、ハクジラ亜目15種、鰭脚類6種、海牛目2種、ラッコ、ホッキョクグマ、および陸上の姉妹群5種)の *cytb* 配列から系統樹を作り、**陸上から海への独立した進出が哺乳類進化で何度起きたか**を観察するテーマ性のあるノートブックです。

- **解析の流れ**(入門編 + 本格編)は `1_birds.ipynb` と同じく、Python 単独の NJ 法 + MAFFT/IQ-TREE による最尤法という二段構え
- **大きな違い: 外群がない**
  - 鳥類セットでは「ミシシッピワニ」を外群として根を決めましたが、本セットは全員が哺乳類のため外群が存在しません
  - そこで **中点ルーティング(midpoint rooting)** — 「最も遠いペアの中点」を根とする — という手法を導入します
- **生物学的考察** — できあがった樹を読みながら、「クジラ・アザラシ・ジュゴン・ラッコ・ホッキョクグマがそれぞれ独立に海洋適応した」という哺乳類進化の壮大なドラマを観察

**学習のねらい**: 系統樹を**読み解く力**を養うことです。鳥類セットで身につけた手法を別のデータに適用し、「ルーティング」という根の置き方の選択肢を学び、樹の形から進化のシナリオを推測するというバイオインフォマティクスの醍醐味を体験します。

## データセット一覧

| パス | 内容 | 件数 |
| --- | --- | --- |
| `phylogenetics/birds/birds_cytb.fasta` | 31種の鳥類(+ワニ)の cytb 配列 | 31 |
| `phylogenetics/birds/birds_cytb_metadata.tsv` | 学名・和名・分類群などのメタ情報 | 31 |
| `phylogenetics/birds/phylopic/` | 各種のシルエット画像と CC ライセンス情報 | — |
| `phylogenetics/mammals/mammals_cytb.fasta` | 37種の哺乳類の cytb 配列 | 37 |
| `phylogenetics/mammals/mammals_cytb_metadata.tsv` | 学名・和名・系統(Mysticeti / Odontoceti …)などのメタ情報 | 37 |
| `phylogenetics/mammals/phylopic_mammals/` | 各種のシルエット画像と CC ライセンス情報 | — |
| `phylogenetics/gpcr/all_GPCRs.fasta` | 25種のヒトGPCR配列(全 Class A) | 25 |
| `phylogenetics/gpcr/gpcr_list.tsv` | 遺伝子名・UniProt ID・クラス・機能注釈 | 25 |
| `phylogenetics/gpcr/pdb/` | AlphaFold予測構造(PDB形式)、ファイル名は `{GENE}_AF-{UniProt}.pdb` | 25 |

FASTAヘッダの書式は独自フォーマットで、位置で意味が決まっています。

- `birds/`, `mammals/`: `>Scientific_name|RefSeq_accession|CYTB`
- `gpcr/all_GPCRs.fasta`: `>GENE|UniProt|ClassA|Description|RefSeq:NP_xxxxx`

PhyloPic 画像ディレクトリには必ず `phylopic_attribution.tsv` が同梱されており、アーティスト名・ライセンスURL・`matched_level`(その種そのもの / 属レベル / 上位分類のシルエット、のいずれか)が記録されています。**画像の追加・削除時はこのファイルも必ず更新してください**(CCライセンス遵守のため)。
